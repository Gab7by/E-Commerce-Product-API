A RESTful API built with Django REST Framework (DRF) for managing products in an e-commerce system.
The API provides user authentication, secure access to product data, and CRUD operations on products.

Features
User registration
User login with token authentication
Token-based access control
Product creation, retrieval, updating, and deletion
Product search by name and category
Product filtering using Django Filters
Secure endpoints for authenticated users only
Django REST Framework Browsable API support
